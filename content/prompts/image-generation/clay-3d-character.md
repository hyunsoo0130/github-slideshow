title: "Clay 3D Character Portrait"
description: "Generate a cute, clean clay-style 3D character portrait in Pixar/toy aesthetic. Great for profile pictures, app mascots, social media avatars, and brand characters."
tool: "nano-banana"
tags: ["3d", "clay", "character", "avatar", "cute", "pixar"]
date: "2026-04-04"
promptText: |
  Create a 3D clay-style character portrait of [CHARACTER DESCRIPTION].

  Art style: Pixar-meets-toy-store aesthetic. Everything is made of soft, slightly
  matte clay material. Smooth surfaces with subtle fingerprint-like texture on the clay.
  Colors are slightly desaturated — warm, friendly, approachable palette.

  Character: [CHARACTER DESCRIPTION]. Expression: [EXPRESSION].
  Face features: Large, slightly simplified eyes (mildly chibi proportions).
  Round cheeks. Small, defined nose. Simple, expressive mouth.
  Skin: [SKIN TONE] smooth clay surface. Hair: [HAIR DESCRIPTION], clay-sculpted texture.

  Clothing: Simple [CLOTHING] — clay texture, clean solid colors.
  Small details like buttons or collar are slightly oversized for readability.

  Lighting: Soft three-point studio lighting. Main light from upper left 45°.
  Warm fill light from right. Top rim light for character separation.
  Result: friendly, even, toy-catalog lighting. No harsh shadows.

  Background: Soft gradient circle behind character — pale [ACCENT COLOR] fading to white.
  OR: Flat muted [ACCENT COLOR] background (specify preference).

  Camera: Bust/portrait framing — head and shoulders. Slight upward angle (5°).
  Aspect ratio: 1:1 or 4:5 portrait.

  Output feel: App store screenshot avatar. Premium toy brand catalog.
  Warm, friendly, instantly likeable.
usage: |
  1. Open Nano Banana
  2. Replace all placeholder variables (see below)
  3. Start with simple descriptions — the more specific, the better the result
  4. Generate 3–4 variations and pick the most expressive one
  5. For product mascots: add your brand color as [ACCENT COLOR]

  This prompt is ideal for: app mascots, Discord profile pictures, Twitch avatars,
  brand characters, children's content, and game character concepts
variables: |
  [CHARACTER DESCRIPTION] → Simple description of the character
    Examples: "young woman with short bob hair", "elderly man with glasses and a warm smile",
              "teenage boy with baseball cap and headphones", "young girl with pigtails"

  [EXPRESSION] → Emotional state
    Options: happy and confident / curious / playful and mischievous / calm and wise /
             excited / shy and sweet / determined / friendly and welcoming

  [SKIN TONE] → Describe in plain terms
    Examples: light peach / warm golden / medium brown / deep mahogany / pale ivory

  [HAIR DESCRIPTION] → Style and color
    Examples: "short wavy black hair" / "long straight platinum blonde" / "curly auburn afro"

  [CLOTHING] → Simple clothing description
    Examples: "hoodie and t-shirt" / "blazer" / "casual sweater" / "polo shirt"

  [ACCENT COLOR] → Brand or theme color for background
    Examples: soft blue / warm coral / mint green / pale lavender
tips: |
  • Keep character descriptions simple — 5–8 words maximum per variable
  • Add "holding [object]" if you want a prop (book, coffee cup, tool relevant to profession)
  • For brand mascots: describe personality traits ("confident tech founder" works well)
  • Add "made entirely of colorful clay, each section a different candy color" for full-clay pop art
  • Request "chibi proportions, head 1/2 of body height" for a more anime-toy hybrid look
  • Specify "no background, transparent" for logo/avatar use — Nano Banana may produce clean results
