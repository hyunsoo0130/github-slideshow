  Create a full-day training workshop slide deck on "[WORKSHOP TOPIC]" for [AUDIENCE] at [COMPANY NAME].
  Duration: [X] hours. Group size: [X] participants. Facilitator: [NAME].

  AGENDA OVERVIEW SLIDE
  Time | Module | Activity Type
  [TIME]: Welcome & Icebreaker
  [TIME]: Module 1 — [TOPIC]
  [TIME]: Break
  [TIME]: Module 2 — [TOPIC]
  [TIME]: Lunch
  [TIME]: Module 3 — [TOPIC]
  [TIME]: Group Activity
  [TIME]: Wrap-up & Action Planning

  WELCOME SLIDE: Facilitator intro, workshop goals, ground rules, logistics (restrooms, phones, breaks)

  ICEBREAKER SLIDE: "[ICEBREAKER ACTIVITY NAME]"
  Instructions: [3-step activity description]
  Time: 10 minutes. Purpose: [WHY THIS ICEBREAKER IS RELEVANT TO THE TOPIC]

  MODULE 1: [TOPIC] (3–4 slides)
  - Concept introduction with visual
  - Key framework or model (diagram description)
  - Real-world example relevant to [INDUSTRY]
  - Reflection question: "Think of a time when..."

  ACTIVITY SLIDE: "[ACTIVITY NAME]"
  Type: [Individual reflection / Pair discussion / Small group / Full group]
  Instructions: Step-by-step, numbered clearly
  Time: [X] minutes
  Debrief questions: 2–3 questions to discuss after

  MODULE 2: [TOPIC] (3–4 slides)
  Same format as Module 1

  MODULE 3: [TOPIC] (3–4 slides)
  Same format as Module 1

  ACTION PLANNING SLIDE
  "My commitments from today:"
  Participant fills in: 1 thing I'll START doing | 1 thing I'll STOP doing | 1 thing I'll CONTINUE doing
  Accountability partner: "Share with the person next to you"

  RESOURCES & FOLLOW-UP
  3 recommended books/resources
  QR code placeholder to digital resources
  "How to reach the facilitator" — email, LinkedIn

  EVALUATION / FEEDBACK
  3-question verbal check-out:
  1. One word to describe today's session
  2. Most valuable takeaway
  3. One thing you'd change
