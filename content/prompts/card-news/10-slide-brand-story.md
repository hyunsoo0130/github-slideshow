promptText: |
  Create a 10-slide card news series for [BRAND NAME]'s brand story on social media.
  Format for Instagram/LinkedIn carousel or PowerPoint export.
  Each slide description includes: layout, visual direction, copy, and design notes.

  SLIDE FORMAT: 1:1 square (1080×1080px) or 4:5 portrait (1080×1350px) — specify [FORMAT].
  Brand colors: Primary [PRIMARY COLOR] | Secondary [SECONDARY COLOR] | Text: [TEXT COLOR]
  Font style: [FONT STYLE — e.g., "clean modern sans-serif", "elegant serif", "bold display"]

  SLIDE 01 — COVER / HOOK
  Background: Full-bleed [PRIMARY COLOR] gradient or brand photo
  Large text overlay: "[HOOK HEADLINE]" — large, bold, centered
  Sub-text: "[BRAND NAME] Story" — small, below headline
  Visual: Strong hero image or bold color block
  Design note: This slide must stop the scroll. Make it visually surprising.

  SLIDE 02 — THE BEGINNING
  Layout: Text-dominant, simple background
  Headline: "It started with a problem."
  Body: "[FOUNDER NAME] noticed that [CORE PROBLEM IN 2 SENTENCES]."
  Visual: Simple illustration placeholder or founder photo
  Bottom accent: "[FOUNDING YEAR]" in large muted typography

  SLIDE 03 — THE FRUSTRATION
  Layout: Full quote card
  Large quote text: "[RELATABLE FRUSTRATION QUOTE about the problem]"
  Below: "Sound familiar?" in italic
  Background: Light [SECONDARY COLOR] tint
  Design note: Make the reader nod — they should see themselves here

  SLIDE 04 — THE TURNING POINT
  Layout: Split — left dark, right light
  Left (dark background): "Then everything changed."
  Right: "[ONE-PARAGRAPH STORY of the turning point/breakthrough]"
  Visual accent: Arrow or spark icon between the two halves

  SLIDE 05 — THE SOLUTION / WHAT WE DO
  Layout: Feature showcase, 3-icon grid
  Headline: "We built [BRAND NAME] to change that."
  3 icons with labels:
  Icon 1: [FEATURE/VALUE 1] | Icon 2: [FEATURE/VALUE 2] | Icon 3: [FEATURE/VALUE 3]
  One sentence below: "[CORE VALUE PROPOSITION]"
  Design note: Clean, confident. This is the product reveal slide.

  SLIDE 06 — SOCIAL PROOF
  Layout: Testimonial card
  Large quotation mark visual element
  Quote: "[CUSTOMER TESTIMONIAL]"
  Attribution: [CUSTOMER NAME], [CUSTOMER TITLE/COMPANY]
  Background: White or very light — let the words breathe

  SLIDE 07 — THE IMPACT / NUMBERS
  Layout: Big numbers, 3-up
  Headline: "The impact so far:"
  Stat 1: "[BIG NUMBER]" — "[WHAT IT REPRESENTS]"
  Stat 2: "[BIG NUMBER]" — "[WHAT IT REPRESENTS]"
  Stat 3: "[BIG NUMBER]" — "[WHAT IT REPRESENTS]"
  Design note: Make numbers huge (180–220pt) — let the data speak visually

  SLIDE 08 — OUR VALUES / WHO WE ARE
  Layout: 3-line list, large elegant typography
  Headline: "We believe in:"
  Line 1: "✦ [CORE VALUE 1]"
  Line 2: "✦ [CORE VALUE 2]"
  Line 3: "✦ [CORE VALUE 3]"
  Background: Brand color or clean white
  Design note: Simple, powerful. Values as promises.

  SLIDE 09 — THE VISION
  Layout: Cinematic full-bleed
  Bold centered text: "[FUTURE VISION STATEMENT — where you're heading]"
  Sub-text: "We're just getting started."
  Background: Dark or dramatic — most visually ambitious slide

  SLIDE 10 — CALL TO ACTION
  Layout: Clean, centered, action-focused
  Headline: "Join us."
  Body: "[2-sentence invitation — what you want them to do]"
  CTA button visual: "[ACTION TEXT]" — e.g., "Visit [website]", "Follow for more", "Try free"
  Social handles or website URL
  Design note: End with clarity — one clear action. Don't confuse with multiple asks.

  Please generate realistic placeholder copy for all 10 slides using [BRAND NAME] context.
  Write the copy as if [BRAND NAME] is a real brand. Make it emotionally compelling.
usage: |
  1. Open Claude with your brand information ready
  2. Fill in all brand variables (name, colors, story details)
  3. Paste the prompt — Claude will write all 10 slides with complete copy
  4. Open Canva, Figma, or PowerPoint
  5. Create one slide per Claude output block
  6. Apply your brand colors, fonts, and real photography

  Export options:
  - For Instagram: Export each slide as 1080×1080 JPG
  - For LinkedIn: Export as 1080×1350 portrait PDF carousel
  - For presentations: Keep as PPTX
variables: |
  [BRAND NAME] → Your brand/company name
  [FORMAT] → "1:1 square" or "4:5 portrait"
  [PRIMARY COLOR] → Main brand color
  [SECONDARY COLOR] → Supporting color
  [TEXT COLOR] → Body text color (dark gray recommended)
  [FONT STYLE] → Typography personality
  [HOOK HEADLINE] → Slide 1 attention-grabbing headline (under 8 words)
  [FOUNDER NAME] → Founder's name (or "our team" if preferred)
  [CORE PROBLEM] → The problem your brand solves
  [FOUNDING YEAR] → Year brand was founded
  [FEATURE/VALUE 1-3] → Key product features or brand values
  [CORE VALUE PROPOSITION] → One-sentence summary of what you offer
  [CUSTOMER TESTIMONIAL] → Real or placeholder quote
  [BIG NUMBER x3] → Key statistics (users, revenue, growth %)
  [CORE VALUE 1-3] → Brand values or principles
  [FUTURE VISION STATEMENT] → Where you're heading
tips: |
  • Ask Claude to "write this in a warmer, more personal tone" if the output feels corporate
  • Add "include a 'before/after' comparison slide as Slide 5.5" for stronger product contrast
  • Request "rewrite Slide 3 as a 'day-in-the-life' scenario instead of a quote"
  • For B2B brands: focus slides 5–7 on ROI and efficiency metrics
  • For lifestyle/consumer brands: focus on emotion, community, and identity in slides 7–9
  • Post all 10 slides in one carousel for 3–5× more reach than single image posts (proven algorithm boost)
